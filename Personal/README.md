# Personal Financial Records

Canonical store for personal financial documents.

## payroll/
Operational employment records.
- paystubs/<employer>/<year>/ — individual pay statements
- w4/<employer>/ — W-4 forms, named by effective date

## taxes/
Tax-related documents.
- w2/ — all W-2 forms (flat, named YYYY-employer.pdf)
- returns/<year>/ — filed federal and state tax returns
